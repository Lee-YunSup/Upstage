# Agent 기반 논문 구독 서비스
### Upstage Document Parse 및 Solar pro 3 활용

AI 분야는 매일 수십 편의 새로운 논문이 쏟아진다.  
이때 나는 초기에 아래와 같은 문제들을 겪었다.

- 관심 분야의 어떤 학회/저널이 있는지 모르겠다
- 어떤 논문을 읽어야할지 판단하기 어렵다
- 영어 논문을 처음부터 끝까지 읽는 데 시간이 많이 든다
- “매일 한 편씩만이라도 꾸준히 읽고 싶다”는 목표를 지키기 힘들다
결국 좋은 논문이 나와도 놓치거나, 논문 읽기가 부담이 되어 학습이 중단되는 문제가 발생한다.  

이 프로젝트는 특정 학회의 논문을 사람이 읽기 쉬운 형태로 정리하여  
매일 정해진 시간에 요약된 논문을 보내주는 구독형 Agent를 만드는 것을 목표로 하였다.

## 어떤 기술이 사용되었을까?
#### - arXiv API
최신 논문을 자동으로 수집

#### - Streamlit
이메일, 학회, 시간 설정을 위한 웹 UI 구성

#### - Upstage Document Parse
PDF를 구조화된 텍스트로 변환

#### - Upstage Chat (solar-pro3)
학회 검색 및 논문을 한국어 요약으로 변환

#### - SMTP 기반 이메일 전송
요약과 원본 PDF를 자동 발송
